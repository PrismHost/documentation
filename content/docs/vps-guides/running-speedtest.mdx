## Windows

<Steps>
    <Step>
        #### Download the Speedtest CLI  
        Go to the official download page:  
        [https://www.speedtest.net/apps/cli](https://www.speedtest.net/apps/cli)

        - Download the version for **Windows** (64-bit recommended)
        - Extract the ZIP file to a folder like `C:\speedtest\`
    </Step>
    <Step>
        #### Open Command Prompt  
        Press `Win + R`, type `cmd`, and press Enter.  
        Then change to the folder where you extracted the speedtest files:
        ```bash
        cd C:\speedtest
        ```
    </Step>
    <Step>
        #### Run the Speed Test  
        Use the following command:
        ```bash
        speedtest.exe
        ```

        This will perform a basic speed test using Speedtest.net servers.
    </Step>
</Steps>

## Linux

<Tabs items={['Debian / Ubuntu', 'CentOS / RHEL / Alma / Rocky', 'Arch', 'Alpine']}>
    <Tab>

    <Steps>
        <Step>
        **Install Speedtest CLI**  
        ```bash
        sudo apt update
        sudo apt install -y curl
        curl -s https://packagecloud.io/install/repositories/ookla/speedtest-cli/script.deb.sh | sudo bash
        sudo apt install speedtest
        ```
        </Step>
        <Step>
        **Run the Speed Test**  
        ```bash
        speedtest
        ```
        </Step>
    </Steps>

    </Tab>

        **Install Using AUR**  
        You can install via an AUR helper like `yay`:
        ```bash
        yay -S speedtest-cli
        ```
        Or manually via:
        ```bash
        git clone https://aur.archlinux.org/speedtest-cli.git
        cd speedtest-cli
        makepkg -si
        ```
        </Step>
        <Step>
        **Run the Speed Test**  
        ```bash
        speedtest
        ```
        </Step>
    </Steps>

        ```bash
        pip3 install speedtest-cli
        ```

## Done

Your test results will include:

- **Download speed**
- **Upload speed**
- **Ping (latency)**
- **Jitter and packet loss** (depending on method used)

These results can help diagnose slow or unstable connections.

If you're experiencing network issues and unsure how to interpret the results, feel free to [contact us](https://prism-host.com/contact) for assistance.
